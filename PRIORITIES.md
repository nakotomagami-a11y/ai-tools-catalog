# Research Priorities

When scanning for new tools, prioritize in this order:

## High Priority
- **Open-source tools** (any category)
- **Coding & Dev** — IDEs, CLI agents, testing, DevOps, code review
- **Security** — pentesting, SAST/DAST, vulnerability scanning, DevSecOps
- **AI/ML Infrastructure** — agent frameworks, RAG, sandboxes, orchestration
- **Marketing & SEO** — automation, SEO, email, social media, content, analytics

## High-Medium Priority
- **Finance & Trading** — AI trading bots, portfolio analysis, market data, crypto tools
- **Data & Analytics** — SQL tools, dashboards, data visualization, BI

## Medium Priority
- **Productivity** — automation, local AI, workflows
- **MCP servers** — dev tools, databases, cloud

## Lower Priority (still include)
- Design, Game Dev, 3D, Audio/Video
